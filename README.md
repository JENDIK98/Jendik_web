# JENDIK_OFC

Oficiální webová stránka JENDIK - básníka a parodisty

## Funkce
- Futuristický design s gradientními barvami
- Responzivní layout pro všechna zařízení
- Sekce: Hero, O mně, Tvorba, Sociální sítě, Kontakt
- Animace a hover efekty
- Kontaktní formulář

## Technologie
- React + TypeScript
- Tailwind CSS
- Vite
- Lucide React (ikony)
